O que é o PriceSimulator?

O PriceSimulator é uma aplicação web desenvolvida em React 18 para auxiliar equipes comerciais e compradores na simulação e análise de precificação de produtos. O sistema recebe uma lista de itens com seus custos base e calcula automaticamente preços de venda com base em parâmetros configuráveis — IPI, frete, margem de lucro e outras taxas — entregando os resultados de forma clara e exportável.

Propósito

Automatizar e padronizar o cálculo de preços de venda, eliminando planilhas manuais e reduzindo erros operacionais em processos de cotação e negociação.

What is PriceSimulator?

PriceSimulator is a React 18 web application built to help commercial teams and buyers simulate and analyze product pricing. The system takes a list of items with their base costs and automatically calculates selling prices based on configurable parameters — IPI tax, freight, profit margin, and other fees — delivering results in a clear and exportable format.

Purpose

Automate and standardize selling price calculations, eliminating manual spreadsheets and reducing operational errors in quoting and negotiation processes.

Como adicionar um novo campo de cálculo
① constants/defaults.js
Adicionar o valor padrão do novo parâmetro.
export const DEFAULTS = {
  ipi: 10,
  freight: 5,
  newParam: 3  // ← aqui
};
② utils/calculations.js
Criar função pura com a fórmula.
export function applyNewParam(
  cost, newParam
) {
  return cost * (1 + newParam/100);
}
③ hooks/useCalculations.js
Chamar a função no hook de cálculo.
import { applyNewParam }
  from '../utils/calculations';
// usar dentro do useMemo/useEffect
④ components/ConfigPanel.jsx
Adicionar slider/input para o usuário controlar o novo parâmetro.
⑤ App.jsx
Propagar o novo estado para useCalculations e para ConfigPanel via props.
Como adicionar um novo formato de exportação
  1. Criar src/utils/xyzGenerator.js com a função geradora (recebe array de produtos, retorna Blob ou string).
  2. Importar e chamar a função em hooks/useExport.js dentro do switch de formatos.
  3. Adicionar o botão/opção correspondente em components/ExportModal.jsx.
  4. Adicionar as strings de UI para PT e EN no hook de idioma.
Adicionando suporte a um novo idioma
  1. Localizar o objeto de traduções no hook de idioma (ou arquivo de constantes).
  2. Duplicar o bloco pt e criar bloco para o novo idioma (ex: es).
  3. Traduzir todas as chaves.
  4. Adicionar a opção no seletor de idioma em Header.jsx.

  1. Create src/utils/xyzGenerator.js with the generator function (receives product array, returns Blob or string).
